# String _ List

## String = 1 List

- String chứa các phần tử là các kí tự liên tiếp
- String hoạt động y như 1 list: Thêm, sửa, truy cập, xóa,...

# List

- Khai báo List: 
```python
list = []
```
- list là 1 list trống

- Thêm phần tử vào list: append
```python
list.append(element)
```
- Truy cập phần tử:
```python
list[index]
```
- Duyệt tất cả phần tử:
```python
for x in list:
    # lam gi do
```
